# Logger
> Basic logging library (Qt based)
<br/>V0.4

# Dependencies 
Qt v5.12.3
Conan v1.42.1
Other: Add Conan remote https://darkfrancisdod.jfrog.io/artifactory/api/conan/public-conan

# Installation
```bash
cd path/to/Logger
mkdir build
cd build
cmake ..
make
sudo cmake --install .
```
